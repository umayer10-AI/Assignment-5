1️⃣ What is the difference between var, let, and const?

Answer: var and let variables can be reassign, but const cannot be reassigned. var variable is accessible outside the block, otherwise let and const are only accessible inside the block.

2️⃣ What is the spread operator (...)?

Answer: The spread operator (...) is used to expand or copy elements or properties of non-primitive variables like arrays and objects.

3️⃣ What is the difference between map(), filter(), and forEach()?

Answer: 1. map() -> transform values and returns a new array.
        2. filter() -> returns a new array based on a condition.
        3. forEach() -> just like normal loop no return values.

4️⃣ What is an arrow function?

Answer: An arrow function is a shorter way to write functions in JavaScript using the => syntax. (ES6 version)
        Example: const sum = (a,b) => a+b;

5️⃣ What are template literals?

Answer: Template literals are backtick (`) strings in JavaScript that allow variable interpolation and multi-line text.
