1️⃣ What is the difference between var, let, and const?

Answer: var and let variables can be reassign, but const cannot be reassigned. var variable is accessible outside the block, otherwise let and const are only accessible inside the block where they are declared.

2️⃣ What is the spread operator (...)?

Answer: The spread operator (...) is used to expand or copy elements or properties of non-primitive variables like arrays and objects.
